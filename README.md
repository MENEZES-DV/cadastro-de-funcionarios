# Sistema de Cadastro de Funcionários

## Descrição
Este projeto é um sistema simples para cadastro, listagem, busca e remoção de funcionários.  
O objetivo é permitir a gestão básica de informações de colaboradores de uma empresa de forma prática e organizada.  

## Funcionalidades atuais
- Cadastrar funcionário (nome, idade, cargo)  
- Listar todos os funcionários  
- Buscar funcionário por ID  
- Remover funcionário por ID  

## Evoluções planejadas
- O projeto será evoluído passo a passo, incorporando melhorias nos seguintes tópicos:
- Na estrutura e organização planeja-se dividir o código em módulos: classes, persistência e interface/controle, salvamento de dados em JSON para a percistência dos dados, melhorias na validação e confiabilidade e criação de funcionalidades extras.
